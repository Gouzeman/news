# News

Учебный PHP-проект: простой сайт новостей с пользовательской частью и административной панелью.

Проект был сделан для практики базовой backend-разработки на PHP: работа с формами, сессиями, авторизацией, базой данных, CRUD-операциями и разделением доступа для администратора.

## Стек

* PHP
* MySQL
* HTML
* CSS
* Tailwind CSS
* JavaScript

## Возможности

* просмотр списка новостей;
* просмотр отдельной новости;
* пагинация списка новостей;
* регистрация и авторизация пользователя;
* сессии для хранения состояния авторизации;
* разделение доступа по роли пользователя;
* административная панель;
* создание новости;
* редактирование новости;
* удаление новости;
* загрузка изображения для новости;
* страница контактов;
* просмотр и удаление сообщений из формы обратной связи.

## Что было реализовано на backend

* подключение к MySQL;
* работа с таблицами пользователей, новостей и сообщений;
* обработка POST-запросов от форм;
* базовая backend-валидация;
* авторизация через сессии;
* проверка роли администратора перед доступом к админ-страницам;
* CRUD-операции для новостей;
* обработка загрузки файлов;
* вывод сообщений об ошибке или успешном действии через сессии.

## Структура проекта

```text
admin/          # страницы административной панели
auth/           # страницы входа и регистрации
images/         # изображения
js/             # JavaScript-файлы
src/            # собранные стили Tailwind CSS
vendor/         # PHP-обработчики форм и действий
index.php       # главная страница со списком новостей
news-full.php   # страница отдельной новости
contacts.php    # страница контактов
```

## Установка и запуск

```bash
git clone https://github.com/Gouzeman/news.git
cd news
```

Проект рассчитан на запуск через локальный PHP-сервер или окружение вроде OpenServer/XAMPP.

Пример запуска через встроенный PHP-сервер:

```bash
php -S localhost:8000
```

После запуска открыть в браузере:

```text
http://localhost:8000
```

## База данных

Для полноценного запуска нужна MySQL-база данных с таблицами пользователей, новостей и сообщений.

Перед запуском необходимо настроить подключение к базе данных в файле:

```text
vendor/db.php
```

Пример параметров:

```php
$connect = mysqli_connect('localhost', 'root', '', 'database_name');
```

## Статус проекта

Учебный проект на чистом PHP. Основная цель — закрепить базовую серверную логику без фреймворка: формы, сессии, авторизация, работа с MySQL и CRUD.

## Что можно улучшить

* добавить SQL-дамп базы данных для быстрого запуска;
* перенести настройки подключения к БД в отдельный конфигурационный файл;
* заменить устаревшее хэширование пароля на `password_hash()` и `password_verify()`;
* привести все SQL-запросы к подготовленным выражениям;
* вынести повторяющиеся части HTML в общие шаблоны;
* добавить `.gitignore` для служебных папок и зависимостей;
* разделить логику обработки запросов и отображение страниц.
